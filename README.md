# Project Atlas

## Overview

Project Atlas is a web-based healthcare verification platform designed to simplify the process of collecting geotagged verification photographs from healthcare facilities.

The platform aims to reduce manual follow-ups, eliminate unnecessary back-and-forth communication and provide trusted location-based evidence for Record Consults.

## Problem

Healthcare facility verification currently relies on manual communication with hospitals and clinics to obtain geotagged photographs proving that a facility is operational. This process is time consuming, requires repeated follow-ups and often leads to delays due to misunderstandings or incomplete submissions.

## Solution

Project Atlas provides healthcare facilities with a secure verification link that allows them to submit the required photographs directly from their mobile device. The platform automatically captures GPS location and organizes submissions for easy review by Record Consults.

## Mission

Provide a seamless geotagged verification process for healthcare facilities and Record Consults by eliminating unnecessary back-and-forth communication.

## Why I Built This

Project Atlas was inspired by a real operational challenge experienced during healthcare facility verification at Record Consults. Rather than building a fictional project, this platform is being developed to solve an existing workflow problem and demonstrate how technology can improve efficiency in real-world administrative processes.

## MVP (Minimum Viable Product)

Version 1 will allow users to:

- Generate a unique verification link
- Capture three required photographs
- Automatically capture GPS coordinates
- Submit verification securely
- View submissions through a simple admin dashboard
